32.Longest Valid Parentheses

Given a string containing just the characters ‘(‘ and ‘)’, find the length of the longest valid (well-formed) parentheses substring.

Example 1:

Input: “(()”
Output: 2
Explanation: The longest valid parentheses substring is “()”

Example 2:

Input: “)()())”
Output: 4
Explanation: The longest valid parentheses substring is “()()”

题解：
根据题意，可以有两种方法求最长合法括号子串长度:

1.借助栈括号匹配求解

    首先设置变量start,表示合法括号串的起始位置，之后遍历字符串，若遇到左括号，那么将当前下标压入栈中，若遇到右括号，若当前栈为空，则将start更新为下一个坐标，否则栈顶元素出栈，此时栈为空，更新最大值$num = max(num, i - start + 1)$ ，否则$num = max(num, i - m.top())$。

2.动态规划

设置变量数组dp，dp[i]表示当前下标前连续匹配的括号的长度。接着考虑状态转移方程。
考虑动态规划的特点：利用已知子问题的解，来求当前要计算的解。最终得到整个问题的解。回归本题，可知在求dp[i]时，dp[0 ~ i-1]均已得知。
易知当 $s[i] == ‘)’$ 时状态才有可能发生转变，而当 $s[i] == ‘(‘$ 时，dp[i]必为0;
对于 $s[i] == ‘)’$ ，由于dp[i-1]已知，可以向左回溯，寻找第一个与s[i]匹配的左括号，即s[i-1-dp[i-1]]，若s[i-1-dp[i-1]] == ‘(‘，则可以得到一个合法括号子串s[i-1-dp[i-1] ~ i]，其长度为dp[i-1] + 2；再加上下标i-1-dp[i-1]以前的合法括号子串长度dp[i-2-dp[i-1]]，得到最终结果dp[i]；否则++i处理之后的字符串。
